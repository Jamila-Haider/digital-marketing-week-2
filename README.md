# Digital Marketing Internship — Week 2

## Week Title
Search Engine Optimization (SEO) & Website Optimization

## Internship Name
Dawood Tech Next Gen / Digital Marketing

## Objectives
Understand SEO fundamentals, learn how search engines rank websites, perform keyword research, optimize web pages, and analyze competitors' SEO strategies.

## Learning Topics
- Introduction to SEO (On-Page, Off-Page, Technical)
- Keyword Research (Search Intent, Short-tail vs Long-tail, Keyword Difficulty)
- On-Page SEO (Title, Meta Description, URL, Headings, Alt Text)
- Technical SEO (Sitemap, Robots.txt, Speed, Mobile, HTTPS)

## SEO Tools Used
- Google Trends
- Ubersuggest
- AnswerThePublic
- Google Search Console

## Practical Tasks Completed
1. Keyword Research Report — 25 keywords for Restaurant & Food niche
2. On-Page SEO Plan — full optimization for a restaurant webpage
3. Competitor SEO Analysis — 2 competitors analyzed
4. Website SEO Audit — basic audit with findings and recommendations

## Key Learnings
- On-Page SEO controls what's on your page; Off-Page builds authority through backlinks
- Long-tail keywords are easier to rank for and convert better
- Search intent determines what type of content to create
- Technical SEO is the foundation, if it's broken, nothing else works
- Meta titles and descriptions directly affect click-through rates
